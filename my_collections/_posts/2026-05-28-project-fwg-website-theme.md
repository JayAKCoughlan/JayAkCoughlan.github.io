I have been volunteering for the Furry Writers' Guild (FWG) since 2021, having been brought on to help support their [current website](https://furrywritersguild.com) on Wordpress and to help design their prospective new website on the [Ghost](https://ghost.org) blogging platform using the CSS framework [Bulma](https://bulma.io).

Because this was volunteer work, the new website theme was worked on outside of my usual work hours and alongside the [Neon Roar](/2022/05/01/project-Neon-Roar.html) game project.

While the theme is based on Ghost's _Casper_ theme, it is highly customized to meet the FWG's needs. For example, there needed to be a directory of registered guild members, but we couldn't use Ghost's built in membership system because all of the registered member details needed to be controlled by the guild's board. So, I built out a system where Admins of the site could make specific pages for each guild member, add all of the usual details, and then set it as a guild-member using a specific page tag and template. This allowed new guild members to be added easily by the Guild's Board members without needing too much technical knowledge:
